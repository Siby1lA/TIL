# 벨로퍼트님의 모던 리액트

## 1장 리액트 입문

### 01 리액트는 어쩌다 만들어졌을까?

- 기존의JS의 DOM을 건드리는 작업은 번거롭고 난잡하다.
- 리액트는 DOM을 다 날려버리고 다시 만들어서 보여준다.
- 메모리에 가상DOM을 만들어서 성능과 속도도 잡음
- 가상DOM과 브라우저DOM을 비교해 변경된 부분만 패치

### 02 작업환경 준비

- Node.js: Webpack, Babled사용하기 위해 기반인 Node.js가 필요
- VSCode : 내가 주로 사용하는 에디터
